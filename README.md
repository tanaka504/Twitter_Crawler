## Twitter API で遊んでみる

### 使い方
tweet_crawler.py と同じディレクトリに config.py を作成し，以下のように設定する．
```
CONSUMER_KEY = '*****************'
CONSUMER_SECRET = '*****************'
ACCESS_TOKEN = '*******************'
ACCESS_TOKEN_SECRET = '*******************'
```

tweet_crawler.py にいくつか関数があるので適宜切り替えて使用する．
```
python tweet_crawler.py
```
